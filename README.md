# LedEnterprise-Technovation-girls

Kodular Project Files

📁 .AIA – Project File
This is the project file of the app created in Kodular. It contains all screens, logic blocks, visual resources, and settings, allowing editing, updating, or customization. To use it, you need to open the file in Kodular, where you can view, modify, or export the app into installable formats.

How to open the .AIA? (To open the project in Kodular)

1. Access the website: https://creator.kodular.io/?locale=pt_BR⁠
2. Log in with a Google account
3. Go to Projects
4. Click on Import Project (.aia) from my computer
5. Select the .aia file
6. Wait for it to load

After that, the project will open with:

Design screen (interface)
Programming blocks

--- 

📦 .APK – Application File
This is the compiled app file, ready to be used on Android devices. To install it, simply download the file on an Android device, allow installation from unknown sources if necessary, and tap to install. After that, the app can be opened and used directly by the end user, without the need for editing or additional tools.

How to install? (Download the .apk file on an Android phone)

1. Open the file
2. If the system blocks the installation, enable the option “allow installation from unknown sources”
3. Click Install
4. After installation, open the app normally
